## 主要修改内容

### 1. 照片预览功能优化
**问题**：在选择照片下载模式下，用户点击照片时仍会触发PhotoSwipe预览弹窗。

**解决方案**：
- 添加`disablePhotoPreview()`和`enablePhotoPreview()`函数
- 在进入下载模式时：
  1. 保存原始PhotoSwipe实例引用
  2. 销毁PhotoSwipe实例
  3. 移除所有照片项的`href`属性，防止点击跳转
- 在退出下载模式时：
  1. 恢复照片项的`href`属性
  2. 重新初始化PhotoSwipe预览功能

**核心代码修改**：
```javascript
// 禁用照片预览功能
function disablePhotoPreview() {
    if (window.lightbox) {
        originalLightbox = window.lightbox;
        window.lightbox.destroy();
        window.lightbox = null;
    }
    
    photoItems.forEach(item => {
        item.setAttribute('data-original-href', item.getAttribute('href'));
        item.removeAttribute('href');
        item.style.cursor = 'pointer';
    });
}

// 启用照片预览功能
function enablePhotoPreview() {
    photoItems.forEach(item => {
        const originalHref = item.getAttribute('data-original-href');
        if (originalHref) {
            item.setAttribute('href', originalHref);
            item.removeAttribute('data-original-href');
        }
    });
    
    if (originalLightbox) {
        // 重新初始化PhotoSwipe
        import('https://cdnjs.cloudflare.com/ajax/libs/photoswipe/5.4.2/photoswipe-lightbox.esm.min.js')
            .then(PhotoSwipeLightbox => {
                window.lightbox = new PhotoSwipeLightbox.default({
                    gallery: '#my-gallery',
                    children: 'a',
                    pswpModule: () => import('https://cdnjs.cloudflare.com/ajax/libs/photoswipe/5.4.2/photoswipe.esm.min.js')
                });
                window.lightbox.init();
            });
    }
}
```

### 2. 根路径访问控制
**问题**：用户直接访问根路径`/`或`http://127.0.0.1:3000/`时，系统显示空白页面或无效页面。

**解决方案**：
- 修改`GetExplorerPageOrFile`控制器函数
- 当检测到`path`参数为`"/"`或空值时，显示友好的通知页面
- 通知内容包括：
  1. 告知用户需要通过分享链接访问
  2. 显示当前访问的路径
  3. 提供使用说明

**核心代码修改**：
```go
func GetExplorerPageOrFile(c *gin.Context) {
    path := c.DefaultQuery("path", "/")
    path, _ = url.PathUnescape(path)

    // 检查是否是根路径访问（没有有效的path参数）
    if path == "/" || path == "" {
        // 显示静态通知页面，告知用户需要通过分享链接访问
        c.HTML(http.StatusOK, "explorer.html", gin.H{
            "message":        "请通过分享链接访问本页面",
            "option":         common.OptionMap,
            "username":       c.GetString("username"),
            "files":          &[]model.LocalFile{},
            "readmeFileLink": "",
        })
        return
    }
    // ... 原有代码继续执行
}
```

## 使用说明

### 对于最终用户
1. **访问方式**：必须通过完整的分享链接访问，例如：
   ```
   http://127.0.0.1:3000/explorer?path=34_1768403334
   ```

2. **下载照片流程**：
   - 点击右下角"下载"按钮进入下载模式
   - 选择要下载的照片（点击照片选中）
   - 点击顶部的"下载选中"按钮开始下载
   - 等待下载完成后自动退出下载模式

3. **注意事项**：
   - 下载模式下无法预览大图
   - 一次最多下载20张照片
   - 下载完成后照片会保存到手机默认下载目录

### 对于开发人员
1. **路由规则**：
   - 控制器：`controller/page-explorer.go`
   - 路由配置：`router/web-router.go`
   - 根路径处理逻辑已更新，显示友好通知

2. **JavaScript功能**：
   - 预览控制：`disablePhotoPreview()` / `enablePhotoPreview()`
   - 下载控制：`toggleDownloadMode()` / `downloadSelected()`
   - 事件绑定：同时支持点击和触摸事件

3. **调试信息**：
   - 页面显示当前访问路径
   - 控制台输出操作日志
